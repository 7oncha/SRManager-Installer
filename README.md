## Instalacija

1. Skini **[Install_SRManager.bat](https://github.com/7oncha/SRManager-Installer/releases/latest/download/Install_SRManager.bat)**
2. Pokreni ga
3. Installer ce automatski skinuti najnoviju verziju i kreirati shortcut na Desktop
4. Gotovo - pokreni **SR Manager** sa Desktopa

> **Napomena:** Windows Defender moze blokirati .bat skriptu. Klikni "More info" → "Run anyway".

## Funkcije

| Funkcija | Opis |
|----------|------|
| **Dashboard** | Status servera u realnom vremenu, broj igraca, mapa |
| **Modovi** | Automatska provjera i download svih modova sa servera |
| **Join Server** | Jednim klikom - provjeri modove, postavi password, pokreni igru |
| **Admin Password** | Automatski kopira admin password u clipboard pri ulasku |
| **Auto-Update** | Automatski provjerava i instalira nove verzije |
| **Postavke** | Intro scene, developer console, putanje igre |
| **Admin Panel** | Upravljanje serverima - zastiteno admin lozinkom |
| **Multi-Server** | Podrska za vise servera s brzim prebacivanjem |

## Auto-Update

Nakon instalacije, aplikacija se **sama azurira**:

1. Launcher pri pokretanju provjerava ima li nova verzija na GitHubu
2. Ako ima, pojavi se zeleni banner **"Nova verzija dostupna!"**
3. Kliknes **"Azuriraj"** → skine se nova verzija, zamijeni stara i restarta app
4. Ne treba ponovo pokretati installer
