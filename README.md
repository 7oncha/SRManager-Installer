# SR Manager - Slavonska Ravnica Launcher

Launcher za Farming Simulator 25 dedicated server **Slavonska Ravnica**.

## Instalacija

1. Skini **[Install_SRManager.bat](https://github.com/7oncha/SRManager-Installer/releases/latest/download/Install_SRManager.bat)**
2. Pokreni ga (desni klik → "Run as administrator" ako treba)
3. Installer ce skinuti najnoviju verziju i kreirati shortcut na Desktop

## Sto radi SR Manager?

- **Dashboard** - status servera, broj igraca
- **Modovi** - automatska provjera i download modova sa servera
- **Join Server** - jednim klikom postavlja modove, password i pokrece igru
- **Auto-Update** - automatski provjerava i instalira nove verzije
- **Admin Panel** - upravljanje serverima (zastiteno lozinkom)

## Auto-Update

Nakon instalacije, aplikacija se **sama azurira**. Kad izadje nova verzija, prikazat ce se banner i jednim klikom ce se skinuti nova verzija i restartati aplikacija.

## Tehnicke Informacije

- Native Windows WPF aplikacija (.NET 8)
- Single-file .exe (~155 MB, self-contained)
- Ne treba nikakav runtime ni PowerShell
- Nema AV false positive-a
